German Dictation Apps

Three static browser apps for slow German sentence playback. Each one uses an in-browser German TTS model, includes built-in sample sentences, and stores custom sentences in IndexedDB.

Variation 1: Reading Room Warm card layout with sentence browsing and compact controls. calm / library feel Variation 2: Dictation Deck Focused playback panel with a queue-style sentence list. studio / focus mode Variation 3: Language Ledger Notebook-like split view with richer metadata and browsing. editorial / workspace
Quick start

Start a local server in this folder with python3 -m http.server 8000, then open http://localhost:8000/.

Direct example links: App A, App B, App C

Use a local static server when testing so IndexedDB behaves consistently, for example: python3 -m http.server 8000 in this folder. The first playback will also download the German model files into the browser cache.
